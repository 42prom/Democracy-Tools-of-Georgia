# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take security seriously. If you discover a vulnerability in **Democracy Tools of Georgia (DTG)**, please **DO NOT** open a public issue.

### 🛑 Responsible Disclosure

1.  **Email**: Send details to `security@42prom.com` (or open a private Draft Security Advisory if you have access).
2.  **Details**: Include full steps to reproduce the vulnerability (PoC).
3.  **Response**: We acknowledge all reports within 48 hours.

### 🛡️ Scope

- **Backend API** (`/src/routes`, `/src/middleware`) -> High Priority
- **Biometric Service** -> High Priority
- **Smart Contracts** (if applicable) -> Critical Priority

### ⚠️ Out of Scope

- Social engineering attacks.
- DDoS attacks (we rely on infrastructure mitigation).
- Vulnerabilities in third-party libraries (unless actionable by us).

Thank you for helping keep Georgian democracy secure! 🇬🇪
